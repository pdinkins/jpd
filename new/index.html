about-me:$|

My name is Parker Dinkins and I have been playing with computers for over fifteen years. I started getting serious about learning how to code in 2017.


I started by learning HTML and CSS so that I could create websites. I then went on to learn Python which is my favorite language. I learned Javascript in an internet programming class and learned that I really love Python after learning C++ for a Data Structures and Algorithms class.


I consider myself a hacker and a futurist. I love exploring topics like data sovereignty, crypto-currencies and the metaverse. I'm interested in studying how humans interact with technology and how technology interacts with humans. There is a clear two way relationship between technology and people and I hope to shed some light on the subject throughout my lifetime by documenting my journey as I go down the rabbit hole.
